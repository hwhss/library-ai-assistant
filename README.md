# 图小助 - 内师大图书馆专属AI助手

## 项目介绍
本项目是为内蒙古师范大学「智慧图书馆 未来由我造」AI应用创新大赛开发的全功能AI助手，一站式解决图书馆的所有痛点！

## 解决的大赛痛点
✅ **问题没人答**：智能双语问答助手，随时在线解答所有问题
✅ **找书找不到**：不用切换标签，直接在聊天里查馆藏位置
✅ **座位抢不到**：实时查看剩余空位，不用跑空
✅ **资源用不起来**：新书推荐，让好书被更多人看到
✅ **通知总是错过**：到期提醒，帮你算还书时间，再也不会逾期

## 独家特色
🇲🇳 **中文/蒙语双语支持**：内师大专属，支持蒙语提问，本地化服务
🔍 **智能问答**：不用切换标签，直接在聊天里问所有问题，打错字也能识别
⚡ **零依赖**：不用装复杂的包，直接就能运行

## 本地运行
1. 安装Python 3.8+
2. 安装streamlit：`pip install streamlit`
3. 运行：`streamlit run app.py`
4. 打开浏览器，访问 http://localhost:8501 就能用了！

## 部署到公网
你可以直接部署到Streamlit Cloud
1. 把代码传到GitHub
2. 打开 https://share.streamlit.io/
3. 关联你的GitHub仓库，选择app.py，一键部署！
4. 访问 https://hwhss-library-ai-assistant-app-ui26mv.streamlit.app/
